eZecool Learn is an online learning management system. Here students and teachers are combined together for sharing knowledge through a structured course based system. Teachers or instructors can create unlimited number of courses, upload videos and documents according to their expertise and students can enroll to these courses and make themselves skilled anytime and from anywhere. Some are the core featurs with eZecool Learn like Advanced course manager, Drag n drop lesson organizer, Instructor dashboard, Student dashboard, Intutive course player, Quizzes - sections and lessons, Student - Instructor messaging, Course Filters, Multiple language support.

Admin featurs : Setup the portal with all settings, Create course - sell, Allow instructor to sell courses

Student featurs : Create account and browse courses, Save favourite courses and create wishlist, Add courses to cart, Messaging with course instructor

Instructors featurs : Create course from instructor panel, Course approval pending screen, Student purchase course notifications, Admin pays courses price dashboard with history
